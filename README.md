# Profile VAE

This repository contains the implementation of Profile VAE (PVAE) for Genotype Induced Drug Design.

## Structure
- `eb_jepa/`: EB-JEPA related code.
- `ijepa_model.py`: I-JEPA model implementation.
- `vit_vae.py`: ViT-VAE model implementation.
- `interpret_vit_vae.py`: Interpretation scripts.
- Training scripts: `ijepa_train_script.py`, `mse_vit_vae_train_script.py`, `vit_vae_train_script.py`.
